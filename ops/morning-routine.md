# Morning Routine

1. `cd ~/work/betterworks-team-os && claude`
2. `/standup` — get your update draft.
3. Skim Slack/Jira for what changed overnight; ask Claude to summarize anything noisy (subagent).
4. Open today's main task. Plan mode (`Shift+Tab Shift+Tab`). Tell Claude to check `/knowledge/<relevant-domain>/` before drafting. Load specific files with `@`.
5. If a real choice comes up (architecture, pricing, scope), tell Claude to consult `/decisions/` first and log a new entry if needed.
6. Exit plan mode and execute.
7. `/compact` after each major task. `/clear` when shifting topics.
8. End of day: capture decisions, doc updates, and new insights. Tell Claude to extract any new patterns into `/knowledge/<domain>/`. **This is the flywheel.**
