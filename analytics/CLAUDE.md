# Analytics

## What's here
- `metrics-definitions.md` — canonical metric definitions, audited by analytics team
- `queries/` — saved SQL by metric or area
- `playbooks/` — verified investigation flows ("how to debug a drop in DAU")

## Rules
- Always cite the file you got a definition from
- Don't invent a query; check `queries/` first or ask before writing one
- Never hallucinate a metric definition — if it's not in `metrics-definitions.md`, say so
