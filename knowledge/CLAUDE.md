# Knowledge

## What's here
Domain-specific learning folders. Each contains:
- `rules.md` — confirmed patterns (promoted after 3+ confirmations)
- `hypotheses.md` — untested patterns
- `knowledge.md` — raw observations

## Domains
- `pricing/` — pricing patterns, buyer signals
- `onboarding/` — onboarding friction, aha moments
- `competitors/` — competitive intelligence
- `metrics/` — metric interpretation patterns

## Rules
- Check `INDEX.md` first to find the right domain
- Promote hypothesis → rule only after 3+ independent confirmations
- Demote rule → hypothesis if contradicted by new data
